# C20s Discord
Sends a custom discord invite link

### Commands
|**Command**|**Description**|
|-----------|---------------|
|/discord   |Sends a discord invite link|

### Features
- [X] PM4 Support
- [X] Fully customizable message

### Config

message: "§7{§bServer§6Name§7}§r - Invite link §bdiscord.gg/yourCode"
